# Verity Voice Cloner - Fish Audio API Demo

A web application that uses the Fish Audio API to generate speech using the Verity voice model.

## Features

- 🎙️ Generate speech using the Verity voice model
- 🔑 Secure API key storage in browser
- 🎵 Audio playback and download
- 📱 Responsive design for all devices

## Setup

1. Get your API key from [Fish Audio](https://fish.audio)
2. Open the website and enter your API key
3. Type your text and click "Generate Voice"

## Voice Model

- **Name:** Verity
- **Voice ID:** 8d21b053e2804e2a890e1cf62f267b6f
- **Source:** [Fish Audio](https://fish.audio/m/8d21b053e2804e2a890e1cf62f267b6f/)

## Technical Details

- Uses Fish Audio REST API
- Pure HTML, CSS, and JavaScript
- No backend required
- API key stored in browser localStorage

## GitHub Pages Deployment

1. Fork this repository
2. Go to Settings > Pages
3. Select your main branch as source
4. Your site will be available at `https://yourusername.github.io/repository-name/`

## Security Note

Your API key is stored locally in your browser and is never sent to any server other than Fish Audio's API.
